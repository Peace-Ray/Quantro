This Update — 0.8.2: Customization

  • Gesture controls: use swipes and gestures to control piece movement instead of onscreen buttons. Pick your preferred control style in the Settings menu.
  • Custom control options: configure as you like.
  • Design a game: create your own polyomino puzzle game that includes Trominoes, Tetrominoes or Pentominoes (3, 4 or 5 blocks each) within a custom-sized game field. Single player only for this update.
  • Show "Next" and "Reserve" piece previews exactly as they will enter the game, or rotated 90 degrees for a better fit. Configure in the Settings menu.
  • Automatically pause the game when the Menu button is pressed.
  • Better graphics on big-screen, high-RAM devices.
  • Internet multiplayer closed beta is over; fewer permissions as a result. Internet multiplayer returns as open beta soon. User agreement and privacy policy updated to reflect this change.

Next Update — Internet Multiplayer Open Beta

  • Create public Internet lobbies without sending a text.
  • Invite a friend to a private Internet lobby over SMS, Email, Facebook or Google Plus.
  • No device restrictions: Internet multiplayer available even on tablets without text messaging.
  • Fewer permissions: Internet multiplayer will no longer require Send/Receive SMS, Read Contacts or Boot Completed permissions.
  • Better netcode.
