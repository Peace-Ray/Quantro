Join or create a local WiFi Lobby. Tap a lobby to join it; long-press to examine lobby details. Tap the 'X' to remove a lobby from the list.

Lobbies available on your WiFi network are automatically added to the lobby list.

WiFi Multiplayer requires an active connection to a local WiFi network.

Quantro's multiplayer features are currently in BETA. Availability is not guaranteed; you may experience bugs or connection failures.


Create Lobby

Create a new lobby for WiFi Multiplayer. Hosting a lobby requires a reliable WiFi connection.


Find Lobby

Locate a WiFi lobby by its IP address. When a lobby is available but has not been automatically discovered, it can be manually added to the lobby list.


Unhide Lobbies

Restore any available lobbies that have been removed from the list.

Troubleshooting

Quantro will automatically detect WiFi lobbies available on your local network. In certain cases, however, automatic detection can fail.

If you know a WiFi lobby is available but Quantro does not detect it, try these steps to make the lobby appear.

  • Check that both devices are connected to the same WiFi network.
  • Ensure that the hosting device's screen is on, with the Quantro lobby displayed.
  • Try manually adding the lobby's IP address to the lobby list, using either an Android Beam™ invitation from the host or the "Find Lobby" button.
  • (Advanced) Check your network configuration to ensure direct communication is allowed between devices, without port forwarding or network address translation.
  • (Advanced) Check your network configuration to ensure that UDP Broadcast is supported — some routers/gateways do not forward Broadcast messages by default.
